# Contributing

Thank you for considering contributing to Strapi Redirect Manager!

## Getting Started

1. Fork the repository and clone your fork.
2. Install dependencies: `npm install`
3. Start the plugin in watch mode: `npm run watch:link`

## Development

- **Build:** `npm run build`
- **Type check (server):** `npm run test:ts:back`
- **Type check (admin):** `npm run test:ts:front`
- **Unit tests:** `npm test`

## Pull Request Guidelines

1. Create a feature branch from `main`.
2. Keep commits focused — one logical change per commit.
3. Follow existing code style: TypeScript strict, 2-space indent, single quotes.
4. Add or update tests for any new functionality.
5. Ensure `npm test` passes before submitting.
6. Write a clear PR description explaining what changed and why.

## Reporting Bugs

Open an issue at [GitHub Issues](https://github.com/OverProgresser/strapi-redirect-manager-plugin/issues) with:
- Strapi version
- Plugin version
- Steps to reproduce
- Expected vs actual behavior
